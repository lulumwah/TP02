1-¿qué es una URL?

Una URL es una cadena de texto unica que se puede encontrar en alguna pagina de internet, este tambien es conocido como web o enlase

2-¿qué es una URL absoluta?

Una URL absoluta se utiliza para enlazar a recursos en otros sitios web y proporciona una dirección completa y única para el recurso

3-¿qué es una url relativa?

Una URL relativa localiza un recurso utilizando una URL absoluta como punto de partida

4-¿cuál es la diferencia entre una url relativa y una url absoluta?

Una dirección URL absoluta contiene toda la información necesaria para localizar un recurso. Una dirección URL relativa localiza un recurso mediante una dirección URL absoluta como punto de partida.

10-¿Cuál es el propósito de la etiqueta anchor?

El proposito de la etiqueta anchor o ancla es crear un enlace a otras páginas de internet, archivos o ubicaciones dentro de la misma página, direcciones de correo, o cualquier otra URL.

11-¿Qué atributo se utiliza en la etiqueta anchor para especificar la URL a la que debe enlazar?

El atributo que se utiliza en la etiqueta anchor para especificar la URL es "href"

12-¿Qué sucede si el atributo "href" en la etiqueta anchor está vacío?

Lo que susede es que no te lleva a ningun enlase o URL

13-¿Cuál es el uso del atributo "target" en la etiqueta anchor y cuáles son los posibles valores que puede tener?

se puede colocar en un enlace para indicar que la página de destino hacia la que apunta se abra en otra ventana del navegador, los valores que puede tener este son, _self , _top , _parent , _new
